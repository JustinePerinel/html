Les extensions d'images

.tiff
image pour le print ou la publication papier
.jpeg, .jpg
Image compressée définitive ; pour les photographies, sans transparence
.gif
Image compressée définitive ; pour les dessins et les images animées avec transparence
.bmp
Image compressée non définitve; image bitmap
.png
Image compressée pour les photographies, les dessins ; png-8 sans transparence et png-24 avec transparence
.svg
Image vectorielle, transparence
.ico
Image pour les méta-données, elle prend place dans la partie head
